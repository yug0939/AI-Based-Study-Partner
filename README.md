# AI-Based-Study-Partner

A Python application that helps students track study sessions and generate personalized quiz questions.

FEATURES

- Log Study Sessions: Record subject, duration, topics, and notes
- Generate Quizzes: AI-powered questions based on your study topics
- Study Analytics: Track study time, streaks, and get recommendations
- View History: Browse all past study sessions
- Data Persistence: Automatically saves your progress

SUBJECTS SUPPORTED

- Math: Arithmetic problems and calculations
- Science: Countries, scientific concepts, discoveries
- History: Historical events and concepts
- Programming: Technical terms and explanations

QUICK START

1. Run the application:
   python ai_study_partner.py

2. Choose from the menu:
   - Press 1 to log a study session
   - Press 2 to generate a quiz
   - Press 3 to view analytics
   - Press 4 to view study history
   - Press 5 to exit

HOW TO USE

1. Log your study sessions after studying
2. Use the quiz feature to test your knowledge
3. Check analytics for study insights
4. Follow recommendations to improve habits

DATA STORAGE

- Study sessions are saved in study_sessions.json
- No installation required - uses Python standard library
- Your data persists between sessions

REQUIREMENTS

- Python 3.6 or higher
- No external dependencies

TIPS

- Study consistently to build streaks
- Use different subjects for balanced learning
- Take quizzes regularly to reinforce knowledge
- Review your analytics to track progress
